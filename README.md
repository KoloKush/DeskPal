# DeskPal
Raspberry Pi Zero LCD 16x2 Interactive Desk Screen

Raspberry Pi Zero W used for this project running 32 bit Raspbian Lite.
I have attached a picture of my setup. And a general schematic from Adafruit's webpage.

I have added a text document with the linux commands I ran.
MainProgramV3.py is the initial release of the startup python. Improvements will be made.

The program will autostart using crontab running a launch.sh file that runs the python3 program.
